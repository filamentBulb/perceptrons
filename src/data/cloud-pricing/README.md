# Cloud Pricing Dataset

Generated by:

```bash
python scraper.py --pricing-estimates --gcp-api-key-file=.gcp_api_key
```

Files:

- `inputs.json`: editable usage quantities, provider defaults, and regions.
- `catalogue.json`: normalized public unit prices from official APIs/docs.
- `estimate.json`: monthly budget model, provider totals, service totals, line items, warnings, and scenarios consumed by the UI.

The GCP API key should stay outside git. Use `.gcp_api_key`, an environment variable, or `--gcp-api-key-file`.
